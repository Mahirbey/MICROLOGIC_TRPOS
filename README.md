# MICROLOGIC_TRPOS

这是一个软件项目，具备土耳其地方特色POS系统以及相关业务的管理系统。
该系统由微软公司的C#来编写。研发团队需要8人以及数名业务人员。
开发所用框架可以使用美国阿帕奇软件基金会工程。



开发与上线流程包含4大阶段：

（1）需求分析和功能分析已经确切定位
（2）研发各个模块
（3）产品专员以及相关人员的测试
（4）正式推广
